# Smart Resume Builder

A modern AI-powered Resume Builder web application that helps users create, analyze, and improve resumes effortlessly.
This project combines a clean UI with intelligent backend features to generate professional resumes and provide AI-driven feedback for better job applications.

---

## 🚀 Features

- Create professional resumes easily
- Add personal information, education, experience, and skills
- Clean and responsive UI
- Dynamic resume preview
- Download or export resume
- Analyze resume content using AI 
- Get suggestions for improving content quality and enhancing keywords
- Better formatting tips
- Helps optimize resumes for ATS (Applicant Tracking Systems)

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/resume-builder.git
```

### 2. Navigate to the project folder
```bash
cd resume-builder
```
### 3. Install backend dependencies
```bash
cd backend
npm install
```
### 4. Install frontend dependencies
```bash
cd frontend
npm install
```
### 5. Start Backend
```bash
cd backend
npm start
```
### 6. Start Frontend
```bash
cd frontend
npm start
```

### 7. The application will run on:

Frontend: `http://localhost:3000`  
Backend: `http://localhost:5000`

---

## 🔐 Environment Variables

Create a `.env` file inside the **backend** folder and add:
```bash
PORT=5000
MONGO_URI=your_mongodb_connection_string
GROQ_API_KEY=your_grok_api_key
```
